# User Story: Jokes Website

## Title
A Simple and Fun Jokes Website

## As a
Visitor to the website

## I want to
View a collection of jokes, get a random joke, and browse jokes by category

## So that
I can have a fun and entertaining experience online

## Acceptance Criteria
- The homepage displays a welcome message and a featured joke
- There is a button to get a random joke
- Jokes are organized by categories (e.g., puns, knock-knock, one-liners)
- Users can browse jokes by category
- Each joke has a setup and punchline (or main text)
- The website is visually appealing and mobile-friendly
- (Optional) Users can submit their own jokes for review

## Out of Scope
- User authentication
- Moderation or admin panel
- Persistent user accounts

## Notes
- The site should be easy to navigate and quick to load
- Jokes can be hardcoded or loaded from a simple file/database
- Focus on a delightful, simple user experience
