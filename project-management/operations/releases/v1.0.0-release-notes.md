# Release Notes: Startup Idea Roaster v1.0.0

**Date**: 2026-03-22
**Version**: 1.0.0
**Status**: Released

---

## What's New

First release of the Startup Idea Roaster MVP.

### Features

- **Pitch Submission**: Submit a startup idea as a short pitch (1-3 sentences, max 300 chars)
- **AI Roast Score**: Fun 5-level score scale:
  - 💀 Certified Dumpster Fire
  - 🔥 Pivot Needed ASAP
  - 😐 Meh But Possible
  - 💎 Hidden Gem
  - 🦄 Future Unicorn
- **Brutal Honest Feedback**: 2-4 paragraphs mixing sarcastic roast with real critique
- **Fix-It Suggestions**: 3-5 tactical + strategic action items
- **Shareable Links**: Each result has a unique URL — share your roast with one click
- **Anonymous**: No login or account required

### Technical

- React 18 + Vite frontend with TailwindCSS
- Express.js backend with Anthropic Claude claude-sonnet-4-6
- In-memory result store with UUID share links
- Auto port detection (starts from 3001)
- Single-command launch with browser auto-open
